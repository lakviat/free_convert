# FreeConvert Static Site

Static, client-side image converter site for GitHub Pages.

## Structure
- `index.html`: main landing page.
- `assets/`: shared CSS, JS, and logo.
- `*/index.html`: individual converter pages per route.

## Local preview
Open `index.html` in a browser or use any static file server.
